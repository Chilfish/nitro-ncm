### 网易云音乐 api 但 nitro.js

仅实现一些用到的 api，逆向部分全部来自 [Binaryify/NeteaseCloudMusicApi]，仅是用 [nitro] 重写了一遍

### Get Started

```bash
pnpm install

pnpm dev
```
